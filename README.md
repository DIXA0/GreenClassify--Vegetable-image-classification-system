# GreenClassify 🌱

GreenClassify is a deep learning-based vegetable image classification system
built using Convolutional Neural Networks (CNN) and deployed using Flask.

## Features

- Classifies 15 types of vegetables
- CNN-based image classification
- Flask web application
- Image upload and prediction

## Tech Stack

- Python
- TensorFlow / Keras
- Flask
- HTML, CSS

## How to Run

1. Install requirements
2. Train the model using `train.py`
3. Run Flask app using `app.py`

## Dataset

Vegetable Images Dataset (Kaggle)
